Landing page desenvolvida para apresentação de portfólio profissional, reunindo informações pessoais, habilidades técnicas e projetos desenvolvidos durante a formação em desenvolvimento web.

O projeto foi criado com foco em organização de código, responsividade, acessibilidade básica e interatividade, podendo ser utilizado como base para futuras melhorias e novos projetos.

---

##  Versão

v1.0.0 
---

##  Objetivo

Desenvolver uma landing page funcional para apresentação profissional contendo:

- Nome 
- Descrição pessoal
- Habilidades técnicas e comportamentais
- Lista de projetos
- Tecnologias utilizadas
- Links para demonstração
- Layout responsivo
- Interatividade com JavaScript

---

## Funcionalidades

✔ Layout moderno  
✔ HTML semântico  
✔ CSS com Flexbox e Grid  
✔ Responsividade (Desktop / Tablet / Mobile)  
✔ Scroll suave  
✔ Menu ativo automático  
✔ Header dinâmico ao rolar  
✔ Animação ao aparecer  
✔ Organização em seções  
✔ Código modular  

Seções da página:

- Home
- Sobre
- Habilidades
- Projetos
- Contato

Sobre Mim

Desenvolvedora Web Fullstack com experiência no desenvolvimento de aplicações completas utilizando Java, Spring Boot, React e JavaScript. Atuo tanto no front-end quanto no back-end, criando soluções modernas, responsivas e bem estruturadas.

Possuo conhecimento em criação de APIs REST, autenticação com JWT, integração com banco de dados PostgreSQL, desenvolvimento de interfaces com React e aplicação de boas práticas de código, versionamento com Git e metodologias ágeis como Scrum e Kanban.

Tenho interesse em evoluir constantemente na área de desenvolvimento, buscando construir sistemas eficientes, organizados e escaláveis, aplicando conceitos de arquitetura limpa e boas práticas de software.
